# מדריך למשתמש — OR Planer 📐

## מה עושה האפליקציה?

האפליקציה מקבלת שרטוט של פרופילי אלומיניום (תמונה או קובץ JSON),
ויוצרת אוטומטית את כל הקבצים שצריך להזמנה:

- **Excel** — טבלת הזמנה מסודרת
- **PDF סיכום** — טבלה מודפסת עם כל הפנלים
- **PDF שרטוטים** — שרטוט הנדסי לכל פנל בנפרד (חתך, פריסה, מידות)
- **DXF** — קבצי שרטוט לפתיחה ב-AutoCAD

---

## שלב 1: הכנת המידות

### אפשרות א׳ — צילום שרטוט ביד (הכי קל)

צלם את השרטוט בנייד. האפליקציה תחלץ את המידות אוטומטית.

**טיפים לצילום טוב:**

- צלם על רקע בהיר (נייר לבן או משובץ)
- וודא שכל המספרים קריאים וברורים
- אל תחתוך חלק מהשרטוט — צלם את הדף המלא
- אור טוב, בלי צללים על השרטוט
- אפשר לצלם כמה דפים — העלה את כולם ביחד

**מה לכתוב בשרטוט:**

```
לכל פנל/פרופיל, כתוב:

1. מספר מזהה (למשל: S-101, או מספר כלשהו)
2. אורך במ"מ (למשל: 2300)
3. רוחב במ"מ (למשל: 270)
4. כמות (למשל: x6)
5. מידות הפרופיל — החתך של הכיפוף
6. הערות (למשל: ניקוב / בלי ניקוב / לסובב)
```

### דוגמה לשרטוט ברור:

```
S-101  x6
אורך: 1570 מ"מ

   25 ─┐
       │ 20
  170 ─┘
       │ 20
   25 ─┘

פרופיל: 25, 20, 170, 20, 25
```

**חשוב:** המספרים בפסיקים מתארים **רק את חתך הפרופיל** של אותו פנל (פנל אחד = שורה אחת). אורך, רוחב וכמות הם שדות נפרדים. אם אותו מק״ט מופיע פעמיים עם פרופילים שונים, האפליקציה תציג אזהרה.

**המספרים בפרופיל הם מידות החתך מלמעלה למטה:**

```
   ┌── 25 ──┐
   │        │ 20
   └──────────────── 170 ──────────────────┘
                                            │ 20
                                   ┌── 25 ──┘
```

- המספרים הזוגיים (1, 3, 5...) = קטעים אופקיים (ימינה)
- המספרים האי-זוגיים (2, 4...) = קטעים אנכיים (למטה)

## שלב 2: העלאה לאפליקציה

1. פתח את האפליקציה בנייד או במחשב
2. הקלד **מזהה פרויקט** (למשל: GPN-1)
3. לחץ **"העלאת תמונות שרטוט או JSON"**
4. בחר את התמונה/קובץ — אפשר לבחור כמה תמונות ביחד
5. חכה שהאפליקציה תחלץ את הנתונים (כמה שניות)

## שלב 3: בדיקה ועריכה

אחרי שהנתונים חולצו, תראה שתי רשימות לעריכה:

### פרטי הזמנה ✏️
- מזהה פרויקט
- שם לקוח
- שם פרויקט
- תאריך
- צבע RAL (ברירת מחדל: 9011)
- עובי מ"מ (ברירת מחדל: 2)
- מספר הזמנה
- **פריסה חומר (מזהה)** — מומלץ לתיעוד ולדפי פנל
- **מיקום** — מומלץ (אתר / עיר)
- חומר (ברירת מחדל: אלומיניום)

**מלא את מה שחסר!** שדות ריקים יופיעו עם התראה.

**רשימת בקרה לתוצאה עקבית:** [PERFECT_INPUT_CHECKLIST.md](PERFECT_INPUT_CHECKLIST.md) או `PERFECT_INPUT_CHECKLIST.txt` (אותה תיקייה, UTF-8).

### פנלים ✏️
טבלה עם כל הפנלים. אפשר לערוך כל שדה:

| עמודה | הסבר | דוגמה |
|-------|-------|-------|
| מק"ט | מספר זיהוי הפנל | S-101 |
| אורך | אורך במ"מ | 1570 |
| רוחב | רוחב במ"מ (אם ריק — מחושב מהפרופיל) | 260 |
| כמות | כמה יחידות | 6 |
| סובב | האם לסובב את הפנל (Y/N) | N |
| פרופיל | מידות החתך, מופרדות בפסיקים | 25,20,170,20,25 |
| הערות | הערות חופשיות | ניקוב |

**אפשר גם להוסיף שורות חדשות** בלחיצה על ה-"+" בתחתית הטבלה.

## שלב 4: יצירת הזמנה

1. בדוק את ה**סיכום** — מראה את כל הפנלים עם שטח מחושב
2. לחץ **"צור הזמנה"**
3. חכה כמה שניות — הקבצים נוצרים

---

## שלב 5: הורדה ותצוגה

אחרי היצירה תראה:

### כפתורי הורדה
- **📥 Excel** — טבלת הזמנה
- **📥 PDF סיכום** — טבלה מודפסת
- **📥 PDF מלא (חבילה)** — קובץ PDF אחד: סיכום + כל דפי הפנלים (דורש חבילת `requirements` עם `pypdf`)
- **📥 DXF (ZIP)** — כל שרטוטי ה-DXF בזיפ אחד
- **📥 PDF פנלים (ZIP)** — כל שרטוטי הפנלים בזיפ אחד

### תצוגה מקדימה 👁️
לחץ על כל קובץ כדי לראות אותו ישירות באפליקציה:
- **Excel** — מציג את הטבלה
- **PDF סיכום** — מציג את הסיכום
- **PDF פנלים** — מציג כל שרטוט פנל בנפרד, עם כפתורי **הקודם/הבא**
- **DXF** — הורדה בלבד (צריך AutoCAD לפתיחה)

כפתור **"← חזרה לרשימת הקבצים"** מחזיר לרשימה.

## ספריית אלמנטים 📚

שמור פנלים כתבניות לשימוש חוזר:

### שמירה
1. אחרי יצירת הזמנה, פתח **"💾 שמור אלמנטים לספריה"**
2. לחץ **"שמור"** ליד כל פנל, או **"שמור הכל"**
3. הפנלים נשמרים בספריה

### שימוש חוזר
1. פתח **"📚 ספריית אלמנטים"** בראש העמוד
2. לחץ **"הוסף"** ליד אלמנט שרוצים להוסיף להזמנה
3. שנה את הכמות/אורך/מידות לפי הצורך

### גיבוי הספריה
- **ייצא ספריה** — שומר קובץ JSON לנייד (גיבוי!)
- **ייבא ספריה** — טוען קובץ JSON חזרה

**חשוב:** גבה את הספריה מדי פעם! הספריה מתאפסת כשהאפליקציה מתעדכנת.
